# Eumero Monorepo

This repository uses npm workspaces with:

- `ui/`: Next.js storefront app
- `cms/`: standalone Sanity Studio app

## Scripts

- `npm run dev` - run the storefront
- `npm run dev:studio` - run Sanity Studio on port `3333`
- `npm run build` - build storefront
- `npm run build:studio` - build Sanity Studio
